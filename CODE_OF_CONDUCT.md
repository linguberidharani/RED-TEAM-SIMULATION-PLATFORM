# Code of Conduct

## Our Pledge

We are committed to providing a welcoming, respectful, and inclusive environment for everyone contributing to this project.

## Expected Behavior

- Be respectful and professional.
- Use constructive feedback.
- Help others learn.
- Respect different viewpoints.
- Follow ethical cybersecurity practices.

## Unacceptable Behavior

- Harassment or discrimination
- Personal attacks
- Hate speech
- Spam
- Sharing malicious code intended for misuse
- Encouraging illegal cyber activities

## Enforcement

Project maintainers may remove content or restrict participation that violates this Code of Conduct.
